# Справочник по тегам HTML

React-приложение со справочником HTML-тегов. Каждый тег собран в карточку с коротким описанием, теги сгруппированы по смысловым разделам. Есть поиск, избранное, переключатель светлой/тёмной темы и детальная панель по каждому тегу.

## Возможности

- **96 тегов** по категориям: основы документа, текст и форматирование, ссылки, медиа, списки и таблицы, семантика, формы, скрипты.
- **Поиск** по названию тега, описанию и названию категории в реальном времени.
- **Избранное** — отмечайте теги звёздочкой, список сохраняется в браузере.
- **Детальная панель** — нажмите на любую карточку, чтобы открыть боковую панель с полным описанием, списком атрибутов, примером кода, советами и ссылкой на MDN. Закрывается кнопкой ✕, кликом по затемнению или клавишей Escape.
- **Тёмная и светлая тема** — переключатель в шапке, выбор запоминается, при первом визите подхватывается системная настройка.
- **Адаптивность** — корректно работает на мобильных: панель выезжает снизу, сетка перестраивается в одну колонку.

## Запуск

Нужен [Node.js](https://nodejs.org/) версии 18 или новее.

```
# Установить зависимости
npm install

# Запустить дев-сервер
npm run dev

# Собрать production-сборку
npm run build

# Посмотреть production-сборку локально
npm run preview
```

После `npm run dev` откройте адрес из терминала (обычно `http://localhost:5173`).

## Как добавить тег

Откройте `src/data/tags.js` и добавьте объект в массив `tags` нужной категории:

```js
{ name: '<mytag>', slug: 'mytag', single: false, desc: 'Краткое описание.' }
```

- `slug` — уникальный идентификатор, латиница без пробелов. Используется для избранного и как ключ в списке.
- `single: true` — для одиночных тегов без закрывающей пары (`<br>`, `<img>`, `<input>`). На карточке появится пометка «одиночный».

Чтобы добавить полную детальную панель для нового тега, откройте `src/data/tagDetails.js` и добавьте объект с ключом, совпадающим со `slug`:

```js
mytag: {
  fullDesc: 'Развёрнутое описание тега.',
  attrs: [
    { name: 'атрибут', desc: 'Что делает.' },
  ],
  example: `<mytag attr="value">Пример</mytag>`,
  notes: [
    'Совет или важное замечание.',
  ],
  mdn: 'https://developer.mozilla.org/ru/docs/Web/HTML/Element/mytag',
},
```

Чтобы добавить новую категорию, добавьте объект `{ id, title, description, tags }` в массив `categories` в `tags.js` — она автоматически появится на странице.

## Темы

Цвета определены через CSS-переменные в `src/styles/theme.scss`: блок `[data-theme='light']` для светлой и `[data-theme='dark']` для тёмной темы. Меняйте значения там — они применятся ко всем компонентам.

## Шрифты

Подключены с Google Fonts: Space Grotesk (заголовки), Inter (текст), JetBrains Mono (имена тегов и код). Если хотите заменить на локальные файлы — раскомментируйте блоки `@font-face` в `src/styles/fonts.scss` и обновите переменные `--font-display`, `--font-body`, `--font-mono`.
